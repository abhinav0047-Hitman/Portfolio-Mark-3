Smart CCTV Camera

Project Description: A next-generation AI-powered surveillance solution that enhances security with real-time monitoring, intelligent detection, and automated alerts. Unlike traditional CCTV cameras, this system uses computer vision and deep learning to analyze video feeds, detect threats, and reduce false alarms..

Technologies Used: Python, YOLO , openCV , Numpy , pillow, Tkinter

Key Features:

  • Detects movement in the camera's field of view
  • Object detection and classification (e.g., humans, vehicles, animals)
  • Identifies and verifies individuals using facial features
  • Interactive dashboard
  • Model achieved 85% accuracy on test data

Challenges & Solutions: Initially faced issues with overfitting, which was resolved by implementing regularization techniques and feature selection. Also created a custom time-series validation method to better evaluate model performance.

Image Classification

Project Description: Built a convolutional neural network (CNN) to classify images of cats and dogs with high accuracy. The model can be extended to recognize other animals or objects with additional training.

Technologies Used: Python, TensorFlow, Keras, OpenCV, NumPy

Key Features:

  • Custom CNN architecture with dropout layers
  • Data augmentation to improve generalization
  • Transfer learning with fine-tuning
  • Model achieved 97% accuracy on validation set
  • Deployed as a web application using Flask

Challenges & Solutions: Limited training data was augmented using rotation, flipping, and zoom techniques. Implemented early stopping and learning rate reduction to prevent overfitting. Optimized model for both accuracy and inference speed.

Sentiment Analysis

Project Description: Created a sentiment analysis tool that processes customer reviews and classifies them as positive, negative, or neutral. The system helps businesses quickly gauge customer satisfaction and identify areas for improvement.

Technologies Used: Python, NLTK, SpaCy, Transformers, Flask

Key Features:

  • Preprocessing pipeline for text cleaning
  • Multiple model approaches (Traditional ML and BERT)
  • Real-time analysis of streaming data
  • Dashboard for sentiment visualization
  • Model achieved 89% accuracy on benchmark datasets

Challenges & Solutions: Handled sarcasm and context-dependent sentiment by implementing a hybrid model combining lexicon-based and machine learning approaches. Deployed the model with an API endpoint for easy integration with other systems.
